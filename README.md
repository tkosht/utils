# utils


---

## installation

```
pip install git+https://github.com/tkosht/utils.git
```

もしくは、

```
git clone https://github.com/tkosht/utils.git
cd utils && pip install .
```


---

## Usage in Python

just import utils like:

```python
import utils
```
