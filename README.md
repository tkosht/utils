# utils


---

## installation

```
pip install git+https://github.com/tkosht/utils.git
```

---

## Usage in Python

### サンプル構成を作成

- 実行

```
curl -sSf https://raw.githubusercontent.com/tkosht/utils/main/example/make_example.sh | sh
```

- 確認

```
ls run.py conf/params.yml 
conf/params.yml  run.py
```


### サンプルコード(`run.py`) を実行

```bash
python run.py 
a=2, b=7
```
